# MPP-I: Machine Payments Protocol for Inference

MPP-I is a proposed MPP-compatible, inference-native payment profile for metered inference.

It keeps the MPP-compatible challenge, credential, payment, and receipt flow, then defines the inference-specific lifecycle needed for run-scoped authorisation, cumulative grants, signed meter frames, low-credit and drain behaviour, delivered-unit billing, settlement references, and final inference receipts.
