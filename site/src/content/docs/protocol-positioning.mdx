# Protocol Positioning

MPP-I is a specialised protocol within the MPP family.

MPP makes machine payments possible for web resources. MPP-I makes machine payments work for metered inference.

## What MPP-I Adds

MPP-I keeps MPP compatibility for the outer payment flow, then defines inference-specific semantics:

- inference quotes,
- client policy,
- authorisation scoped to a run,
- cumulative vouchers or credit grants,
- signed meter frames,
- low-credit states,
- drain-entry behaviour,
- stop boundary rules,
- settlement references,
- final inference receipts.

## Current Status

The current artifact is a protocol paper, not a final registered standard. Future work will convert the paper into draft protocol specifications and schemas.
