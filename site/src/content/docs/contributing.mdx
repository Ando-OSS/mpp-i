# Contributing

Use GitHub Issues for actionable changes:

- errata,
- missing citations,
- broken links,
- unclear wording,
- scoped technical corrections.

Use GitHub Discussions for open-ended topics:

- protocol design,
- future profiles,
- settlement models,
- wallet behaviour,
- inference-provider integration,
- library and SDK direction.

See the repository `CONTRIBUTING.md` for full contribution rules.
