# Release Process

Stable versions should be published as GitHub Releases backed by tags.

## Initial Release

- Tag: `v1.0.0`
- Release title: `MPP-I Paper v1.0`
- Release assets:
  - `paper/mpp-i-paper-v1.0.pdf`
  - `paper/mpp-i-paper-v1.0.tex`
  - checksum file

## Citation

Before Zenodo is enabled, cite the tagged GitHub release or a permanent commit URL. After Zenodo is enabled, cite the DOI for the archived release snapshot.
