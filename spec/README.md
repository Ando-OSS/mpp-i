# MPP-I Specification

This folder is reserved for future MPP-I protocol specifications.

The current canonical artifact is the approved paper in `paper/`. Future work will turn the paper into draft specifications, schemas, and implementation guidance.

## Current Status

- No normative wire specification is published yet.
- `Paper v1.0` is the current public technical proposal.
- Draft specs should preserve the paper's core model unless a protocol-change proposal is accepted.
