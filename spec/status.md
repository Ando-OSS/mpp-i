# Specification Status

MPP-I is currently a protocol proposal.

## Current Artifact

- `Paper v1.0`
- Publication date: 2026-06-28
- Status: approved public paper

## Future Spec Work

Future specifications are expected to define:

- payment intent/profile naming,
- quote object fields,
- policy object fields,
- run-scoped credit grant semantics,
- meter frame semantics,
- delivery acknowledgement semantics,
- drain and stop state machine,
- final receipt semantics,
- settlement-method mapping,
- transport profile requirements.
